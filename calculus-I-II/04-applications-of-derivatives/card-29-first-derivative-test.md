# Card 29: First Derivative Test

**Section:** Applications of Derivatives

## Front

How does the First Derivative Test work?

## Back

At a critical point $c$:

- If $f'$ changes from positive to negative, $f(c)$ is a local maximum.
- If $f'$ changes from negative to positive, $f(c)$ is a local minimum.
- If $f'$ does not change sign, there is no local extremum.
