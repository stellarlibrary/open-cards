# Card 1: Function basics

**Section:** Precalculus Review

## Front

What is a function, and what are domain and range?

## Back

A function maps each input to exactly one output.

- **Domain**: allowed input values.
- **Range**: possible output values.
- Common restrictions:
  - Denominator cannot be zero.
  - Even root requires nonnegative input.
  - Logarithm requires positive input.

Example:

$$f(x)=\sqrt{x-2}$$

Domain:

$$x \ge 2$$
