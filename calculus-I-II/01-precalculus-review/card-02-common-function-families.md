# Card 2: Common function families

**Section:** Precalculus Review

## Front

What are the most common function families in Calculus I?

## Back

Important function types:

- Polynomial: $x^2,\ x^3,\ x^n$
- Rational: $\dfrac{p(x)}{q(x)}$
- Exponential: $a^x,\ e^x$
- Logarithmic: $\ln x,\ \log_a x$
- Trigonometric: $\sin x,\ \cos x,\ \tan x$
- Inverse trig: $\arcsin x,\ \arctan x$

Calculus uses these constantly for limits, derivatives, and integrals.
