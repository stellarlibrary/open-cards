# Card 54: Area between curves

**Section:** Applications of Integration

## Front

How do you find the area between two curves?

## Back

If upper curve is $f(x)$ and lower curve is $g(x)$:

$$A=\int_a^b [f(x)-g(x)]\,dx$$

If integrating with respect to $y$:

$$A=\int_c^d [x_{\text{right}}-x_{\text{left}}]\,dy$$
