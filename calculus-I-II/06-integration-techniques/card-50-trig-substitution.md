# Card 50: Trig substitution

**Section:** Integration Techniques

## Front

When should you use trigonometric substitution?

## Back

Use trig substitution for expressions involving:

$$\sqrt{a^2-x^2},\qquad \sqrt{a^2+x^2},\qquad \sqrt{x^2-a^2}$$

Common substitutions:

$$x=a\sin\theta,\qquad x=a\tan\theta,\qquad x=a\sec\theta$$
