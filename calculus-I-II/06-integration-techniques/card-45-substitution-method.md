# Card 45: Substitution method

**Section:** Integration Techniques

## Front

When should you use $u$-substitution?

## Back

Use $u$-substitution when the integrand contains a function and its derivative.

Example:

$$\int 2x\cos(x^2)\,dx$$

Let $u=x^2$. Then $du=2x\,dx$. So:

$$\int 2x\cos(x^2)\,dx=\int \cos u\,du=\sin u+C=\sin(x^2)+C$$
